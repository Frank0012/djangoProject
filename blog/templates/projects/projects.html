Matrix Manipulator

A set of programs for matrix manipulation. There are three classes: MatrixInverter, MatrixMultiplier and SimultaneousEquationSolver. MatrixInverter can invert any real invertible matrix. MatrixMultiplier can multiply any multipliable real matrices. SimultaneousEquationSolver uses matrices to solve simultaneous equations where the number of unknowns is equal to the number of equations.
